# Таблица переходов автомата
1. «б» — буквы (латиница).
2. «ц» — арабские цифры.
3. «о» — операторы: `-`, `/`, `*`, `+`, `(`, `)`, `{`, `}`, `<`, `>`, `[`, `]`, `;`.

## Нетерминалы
1. `S` — начальный.
2. `I` — идентификатор.
3. `C` — константа.
4. `R` — обработка «`>`».
5. `K` — обработка «`<`».
